Tamil Nadu Political Parties Sentiment Analysis – Election 2026

YouTube Comment Sentiment Analysis using AI (Gemma-3-4B-IT)

About This Project

This project analyzes public sentiment toward major Tamil Nadu political parties using publicly available YouTube comments. The dataset covers political discussions between 02 March 2026 – 13 March 2026. Using Google's open-source LLM Gemma-3-4B-IT, each comment is classified as:


  • Positive
  • Negative
  • Neutral

The results are aggregated and visualized to understand how public discussions evolve across political parties.

Methodology

Data Collection

YouTube comments were collected from Tamil news channels discussing Tamil Nadu politics. The YouTube Data API was used to retrieve comments.

AI Sentiment Analysis

Each comment was analyzed using Google's LLM model Gemma-3-4B-IT. The model classifies comments into positive, negative, or neutral sentiment.

Data Processing

Comments were filtered using party entity names and aggregated by date.

Visualization

Interactive charts were built using the ApexCharts JavaScript library.
